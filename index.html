Add another video

To add more videos, upload the file to the repository root (for example video2.mp4) then add another <video> block like this:

<video controls>
  <source src="video2.mp4" type="video/mp4">
</video>

You can also embed YouTube videos using an iframe if you prefer hosting externally.
