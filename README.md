# Food Guide Website

A responsive food guide website template built with HTML, CSS, and JavaScript. Perfect for sharing nutritional information, recipes, and healthy eating tips.

## Features

- 📱 Fully responsive design for all devices
- 🎨 Modern gradient design with smooth animations
- 🥗 Food categories section
- 📊 Nutrition facts information
- 🍳 Featured recipes showcase
- 📧 Contact form
- ⚡ Smooth scroll navigation
- 🔍 Intersection Observer for card animations

## Sections

1. **Navigation Header** - Sticky navbar with smooth scrolling links
2. **Hero Section** - Eye-catching introduction with call-to-action button
3. **Categories** - Grid of food categories (Vegetables, Fruits, Proteins, Grains)
4. **Nutrition Facts** - Information about macronutrients and their benefits
5. **Featured Recipes** - Recipe cards with preparation times
6. **Contact Form** - Get in touch section
7. **Footer** - Site information and credits

## How to Use

1. Clone this repository
2. Customize the content in `index.html`
3. Update the colors and fonts in `styles.css`
4. Extend functionality in `script.js`
5. Push to your `username.github.io` repository

## Customization

### Add Your Own Recipes

Edit the recipe cards in the `recipes` section:

```html
<div class="recipe-card">
    <img src="your-image-url" alt="Recipe Name">
    <h3>Your Recipe Title</h3>
    <p>Recipe description here</p>
    <span class="prep-time">⏱️ XX mins</span>
</div>
```

### Change Colors

Update the color variables in `styles.css`:

- Primary color: `#667eea` (purple-blue)
- Secondary color: `#764ba2` (purple)
- Accent color: `#27ae60` (green)
- Dark color: `#2c3e50` (dark blue-gray)
